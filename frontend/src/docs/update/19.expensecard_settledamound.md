# 🧩 Problem Summary (Plain English)

* A **settlement is made**
* The **daily grid reflects it** (resolved / settled)
* BUT the **weekly expense card still shows $0 income**
* Even though money was actually received via settlement

👉 That means **settlements are not being counted as income** in the expense summary.

---

# 🎯 Expected Behavior (Target)

> If a settlement allocates money to days within a week,
> that money **must appear as INCOME** in the corresponding **expense card**.

## 🔍 Why this happened (root cause)

You **correctly separated**:

* daily grid logic
* settlement logic

But the **weekly expense card was still living in the “old world”** where:

> income == collected that same day

Now income can arrive **late**, so summaries must be **time-aware**, not event-aware.

This is a **very mature accounting problem**, not a mistake.
