## 5. Cell Interaction Wireframe

Clicking a cell opens a **small modal or side panel**.

### Wireframe (Modal)

```
┌──────────────────────────────┐
│ Day 2 – March 2026           │
├──────────────────────────────┤
│ Expected: 100                │
│ Collected: [   50   ]        │
│                              │
│ Status:                      │
│  ( ) Full                    │
│  (●) Partial                 │
│  ( ) Missed                  │
│                              │
│ Notes (optional):            │
│ [_________________________]  │
│                              │
│ [ Cancel ]   [ Save ]        │
└──────────────────────────────┘
```

### Rules

- If collected = expected → auto mark **Full**
    
- If collected = 0 → **Missed**
    
- Anything in between → **Partial**

## 7. Previous Months Behavior (Wireframe Rule)

When viewing past months:

- Grid becomes **read-only**
    
- Cells are still clickable for details
    
- No editing allowed
    

Visual cue:

```
🔒 March 2026 (Locked)
```
