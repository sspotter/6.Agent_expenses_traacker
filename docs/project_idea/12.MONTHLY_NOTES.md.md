# 📝 Phase 10: Monthly Notes & Checklist System

---

## 1. Purpose

Provide a **non-financial notes area** per month to:

* Track reminders
* Track pending actions
* Track context behind numbers
* Avoid misusing expenses or payments as notes

This feature must be:

* Optional
* Non-intrusive
* Completely separate from money calculations

---

## 2. Core Concept

Each month can have **multiple notes**, and each note:

* Has text
* Has a checkbox (done / not done)
* Is independent of income or expenses

This acts like a **monthly checklist / memo board**.

---

## 3. UX Entry Point

### Button Placement

Add a button in the month header area:

```
[ February 2026 ]      [ 📝 Notes ]
```

or

```
[ 📝 Monthly Notes ]
```

---

## 5. Behavior Rules

### Notes

* Notes are **month-specific**
* Notes can be added anytime
* Notes can be edited (text)
* Notes can be checked / unchecked
* Order is user-controlled (optional drag)

### Checkboxes

* Checking a note does **not affect any data**
* No auto-calculations
* No financial meaning

---

## 6. Data Model

### MonthlyNote Table

| Field      | Description  |
| ---------- | ------------ |
| id         | Primary key  |
| user_id    | Owner        |
| month      | YYYY-MM      |
| text       | Note content |
| is_checked | Boolean      |
| created_at | Timestamp    |
| updated_at | Timestamp    |

---

## 7. Mutability Rules

| Action          | Allowed |
| --------------- | ------- |
| Add note        | ✅       |
| Edit note text  | ✅       |
| Toggle checkbox | ✅       |
| Delete note     | ✅       |
| Affect income   | ❌       |
| Affect expenses | ❌       |

Notes remain editable even if the month is locked.

---

## 8. Visual Indicators (Optional)

* Show small badge on 📝 Notes button:

  * Number of unchecked notes
* Checked notes appear faded or strikethrough

---

## 9. Export & Reporting Behavior

### PDF Export

* Monthly notes appear on **last page**
* Checked state included
* Read-only snapshot

### CSV Export

Optional file:

```
monthly_notes.csv
```

Fields:

* month
* text
* is_checked

---

## 10. Permissions & Safety

* Notes are private per user
* No cross-month carryover
* No auto-deletion

## 12. What This Feature Is NOT

❌ A task scheduler
❌ A reminder system
❌ A financial record
❌ A substitute for expenses

It is purely **context + memory**.

---

## 13. Guiding Rule

> **If removing notes changes any financial number, it’s a bug.**
